# Weather App

Check the current weather on a specific location 

## Features

1. Current local time and date

2. Temperatures and humidity

3. Wind speed and direction

4. Sunrise and sunset times

5. Latitude, longitude, system unit, city name and country code need to be specified in the config.json file

7. Error handling and loading info

## Installation

1. `git clone https://github.com/YannMazz/weather-app.git`

2. `cd weather-app`

3. `npm install`

4. use the config.json file to specify your data 

5. `npm run dev`
